# Neighborhood Navigators Website

A simple static shopping-style website for Neighborhood Navigators, ready to push to GitHub Pages.

## Pages
- Home: `index.html`
- Shop: `shop.html`
- About: `about.html`
- Contact: `contact.html`
- Demo Cart: `cart.html`

## How to use
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to Settings > Pages.
4. Under Branch, choose `main` and `/root`.
5. Save.

## To accept payments
Replace the Add to Cart buttons with checkout links from Shopify, Stripe Payment Links, Square, or PayPal.

## Contact form
The contact form currently uses a placeholder Formspree link. Replace `YOUR-FORM-ID` in `contact.html` with your real Formspree form ID.
